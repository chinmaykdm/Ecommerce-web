# Ecommerce-web
FastAPI-based e-commerce web application with user authentication, product management, shopping features, and Swagger API documentation. Built using FastAPI, SQLAlchemy, JWT authentication, HTML, CSS, and JavaScript.
# ShopWave

A modern e-commerce web application built with FastAPI.

## Features
- User registration and login
- JWT authentication
- Product management
- Shopping functionality
- SQLite database
- Swagger API documentation
- Responsive frontend using HTML, CSS, and JavaScript

## Tech Stack
- FastAPI
- SQLAlchemy
- SQLite
- JWT
- HTML/CSS/JavaScript

```bash
pip install -r requirements.txt
